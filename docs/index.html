Where did my day
actually go?

daylog auto-logs your Claude Code sessions, GitHub PRs, Jira & Confluence activity, and calendar meetings — then turns the day into a standup you'd actually write yourself. No cloud, no account, no telemetry: it's all plain JSONL in ~/.daylog/.

Why

It's 11:00 am. Notification lights up: "quick standup update for tomorrow?"

A PR review, a flaky-build call that ran 40 minutes with no ticket attached, a hallway conversation explaining the auth flow twice because the first pass didn't land. Real work — none of it written down anywhere. So the update becomes "worked on some stuff, a few meetings, reviewed a PR" — true, and useless.

daylog doesn't ask you to track better — it stops losing the record you're already generating. GitHub, Jira, Confluence, calendar, and Claude Code pull in automatically; anything else is one line. By evening, it's a standup ready to paste into Teams — not a guess reconstructed at 11am, a record.

A day, logged

09:14
auto-capture

A Claude Code session starts logging itself

Time accrues incrementally as you work — even a session left open for days gets split correctly across each day it was actually active.

10:47
manual — one line

dl log "explained the auth flow to a teammate" -d 20

No integration list is ever complete. Fifteen minutes of hallway debugging gets the same one-line treatment as a two-hour coding session.

11:02
github

A reviewed PR lands in the timeline, unasked

Opened, reviewed, and commented-on PRs are pulled automatically from an authenticated gh CLI — no MCP setup required.

12:30
gap-fill

"90 min unaccounted. Run dl fill."

A quiet macOS nudge, not a nag — fill gaps one line at a time, or skip them. Anything under 15 minutes is ignored automatically.

17:45
evening checkpoint

Calendar, PRs, and Jira sweep themselves in

One scheduled pass before the gap-fill reminder fires, so the record is already mostly built by the time you'd normally start reconstructing it.

17:58
dl standup --ai

A standup that reads like you wrote it

No tool-internal labels, no time-of-day leaking through — related entries merge into one bullet about what actually got done. Copy, paste, done.

06:45
morning brief

Yesterday, waiting in Notes.app

Delivered before you open your laptop — no command to remember, no copy-paste, nothing to forget on the way to your first meeting.
